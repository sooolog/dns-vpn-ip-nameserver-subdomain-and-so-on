## 호스트컴퓨터의 이름 호스트명이란 무엇일까

호스트 컴퓨터란, 브라우저에 홈페이지 파일을 제공하는 컴퓨터를, 호스트 컴퓨터 즉, 호스트명이라 한다.
즉, 호스트 컴퓨터에서 브라우저의 요청에 응답하여 홈페이지 파일을 클라이언트의 컴퓨터로 전송해주었다는 의미이다.

-> 그렇다면, www.gabia.com의 경우, 홈페이지 파일을 제공하는 호스트 컴퓨터(웹서버)의 이름, 즉 호스트명은
무엇일까요? 답은 WWW입니다.

* 호스트명은 서버의 운영 주체마다 다르게 설정할 수 있으나 서버의 특징을 나타낼 수 있는 
용어가 되어야 하므로, 전세계 표준 통신 규약에 따라 같은 역할을 수행하는 서버는 명칭이 
동일합니다. 마찬가지로 동종의 서비스를 제공하는 호스트 컴퓨터의 이름도 동일하거나 
유사할 것입니다.

WWW.도메인.com -> 호스트명:www
domain.도메인.com -> 호스트명:domain
whois.도메인.com -> 호스트명:whois
ftp.도메인.com -> 호스트명:ftp
ns.도메인.com(네임서버1) -> 호스트명:ns
ns1.도메인.com(네임서버2) -> 호스트명:ns1
